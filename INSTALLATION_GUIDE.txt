POS NEXT v3.0 - Modern Retail & Restaurant POS - INSTALLATION GUIDE

1. Install (Frappe v15 & v16 compatible, standalone, no extra pip packages)
   cd ~/frappe-bench
   bench get-app pos_next https://github.com/GoldenJustin/pos_next.git --branch main
   bench --site yoursite install-app pos_next
   bench --site yoursite migrate
   bench build --app pos_next
   bench restart

2. Configure Global Settings
   - Open Workspace POS Next -> POS Next Settings
   - Set Business Type: Retail, Restaurant, Supermarket, Wholesale, etc.

3. Configure POS Profile
   - Enable POS Next
   - Set POS Mode: Retail, Restaurant, Supermarket, Wholesale, etc.
   - Enable Table Management, KDS, Customer Display, Cash In/Out, Split Bill, Offline, Hotkeys
   - Set Receipt Template (Modern Retail recommended)
   - Set KOT Printer Map

4. Restaurant
   - Create POS Floor, POS Tables or seed demo
   - bench --site yoursite execute pos_next.seed.seed_demo_for_profile --kwargs "{'pos_profile':'Your Profile'}"

5. KDS
   - Open /kds?pos_profile=YourProfile on kitchen tablet

6. Customer Display
   - Open /customer-display?pos_profile=YourProfile on second screen

7. Reports via Workspace POS Next

If custom fields not showing: bench --site yoursite execute pos_next.seed.setup_after_install
